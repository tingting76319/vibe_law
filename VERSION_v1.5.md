# v1.5 版本規劃 - 法院判決差異分析

## 目標
分析不同法院對同類案件的判決差異

---

## 功能說明

### 1. 各法院判決差異分析
- 輸入：案件類型（如：勞動檢查報告）
- 輸出：各法院的判決傾向

### 2. 地緣產業特性分析
- 分析法院所在地的產業特性
- 與判決結果的關聯

### 3. 判決傾向比較
- 賠償金額比較
- 刑期輕重比較
- 調解率比較

---

## API 設計

### GET /api/graph/court-analysis
分析不同法院的判決差異

**參數：**
- `caseType` - 案件類型
- `court` - 法院名稱（可選）

**回應：**
```json
{
  "caseType": "勞動檢查報告",
  "courts": [
    {
      "court": "桃園地方法院",
      "caseCount": 150,
      "-relianceOnLIR": 90
    },
    {
      "court": "臺北地方法院", 
      "caseCount": 200,
      "relianceOnLIR": 60
    }
  ]
}
```
