# Legal-RAG 專案目標

## 🎯 最終目標

**最短時間找到合適辯護律師 + 提供最佳訴訟策略 = 最佳媒合效果**

---

## 核心功能

### 1. 律師/法官訴訟邏輯分析
- [ ] 分析法官歷史判決模式
- [ ] 理解不同法院的判斷標準
- [ ] 提取法官裁判風格
- [ ] 律師風格分析

### 2. 智能律師匹配
- [ ] 當事人上傳訴訟內容
- [ ] AI 分析案件類型與難度
- [ ] 根據法官/法院匹配合適律師
- [ ] 律師過往勝率分析

### 3. 訴訟策略生成
- [ ] 律師上傳訴狀/個案資料
- [ ] 分析受審法官過往判例趨勢
- [ ] 產生符合律師風格的訴狀
- [ ] 開庭前策略建議
