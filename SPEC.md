# 台灣法律專家知識系統 (Legal-RAG)

## 1. 專案概述

- **專案名稱**：Legal-RAG (台灣法律專家知識系統)
- **類型**：知識庫檢索系統 + 生成式 AI 應用
- **核心功能**：基於台灣司法院公開判例，提供法律問題問答服務
- **目標用戶**：律師、法務人員、一般民眾

## 2. 功能規劃

### 2.1 核心功能

| 功能 | 說明 |
|------|------|
| 🔍 判例搜尋 | 透過關鍵字搜尋司法院公開判例 |
| 💬 智慧問答 | 使用 RAG 技術回答法律問題 |
| 📚 法規查詢 | 民法、刑法、行政法等法規查詢 |
| 📋 相似案例 | 找出與案情相似的過往判例 |
| ⭐ 收藏功能 | 收藏重要判例方便日後查看 |

### 2.2 資料來源

- 司法院公開判例 API（待串接）
- 全國法規資料庫
- 法務部法規檢索系統

### 2.3 RAG 系統架構

```
┌─────────────┐     ┌──────────────┐     ┌─────────────┐
│   使用者     │ ──> │   檢索系統    │ ──> │   生成系統   │
│  輸入問題    │     │  (向量資料庫)  │     │  (LLM)      │
└─────────────┘     └──────────────┘     └─────────────┘
                           │
                    ┌──────┴──────┐
                    │  法律知識庫   │
                    │ (判例+法規)  │
                    └─────────────┘
```

## 3. 技術規格

### 3.1 技術棧

- **前端**：HTML5 + CSS3 + JavaScript
- **後端**：Node.js / Python (可擴展)
- **向量資料庫**：本地 JSON (可擴展至 Pinecone/Chroma)
- **RAG 引擎**：簡單關鍵字匹配 + 向量相似度

### 3.2 檔案結構

```
legal-rag/
├── index.html           # 主頁面
├── css/
```

## 6. 資料格式

### 判例資料格式
```json
{
  "id": "109台上1234",
  "court": "最高法院",
  "year": 109,
  "caseNumber": "台上字第1234號",
  "type": "民事",
  "title": "車禍損害賠償",
  "summary": "原告請求被告賠償車禍損失...",
  "result": "上訴駁回",
  "relatedLaws": ["民法第184條", "民法第213條"],
  "date": "2020-05-20"
}
```

### 法規資料格式
```json
{
  "id": "civil-184",
  "name": "民法第一百八十四條",
  "category": "民法",
  "content": "因故意或過失，不法侵害他人之權利者...",
  "lastAmended": "2023-01-01"
}
```

---

## 7. 注意事項

1. **資料來源**：所有判例資料來自司法院公開資料
2. **法律聲明**：本系統僅供參考，不構成法律意見
3. **API 金鑰**：正式上線需申請司法院 API
