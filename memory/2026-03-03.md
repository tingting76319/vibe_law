### 2. 法規資料庫串接 ✅

**新增檔案：**
- `backend/src/db/lawsMigration.js` - 法規資料表 Migration
- `backend/src/repositories/lawRepository.js` - 法規 Repository
- `backend/src/services/lawService.js` - 法規 Service
- `backend/src/routes/laws.js` - 法規 API 路由
- `docs/LAWS_API_RESEARCH.md` - 研究文件

**API 端點：**
| Method | Endpoint | 說明 |
|--------|----------|------|
| GET | /api/laws/search | 搜尋法規 |
| GET | /api/laws/:lawId | 取得法規詳情 |
| GET | /api/laws/category/:category | 依類別取得法規 |
| GET | /api/laws/categories | 取得所有類別 |
| GET | /api/laws/stats | 取得統計 |
| POST | /api/laws/import | 匯入單一法規 |
| POST | /api/laws/batch-import | 批次匯入法規 |

**法規資料表 (laws)：**
- law_id, law_name, law_category, chapter, section, article
- content, effective_date, promulgate_date
- related_laws, keywords
