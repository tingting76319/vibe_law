# Vibe Law UI/UX 重新設計方案

## 設計目標

1. **全版面設計** - 增大可視範圍
2. **功能整合** - 相似類別合併到同一 Tab

## 重新規劃 Tab（5個）

### 合併邏輯

| 新 Tab | 合併來源 | 功能 |
|--------|----------|------|
| 🔍 智慧搜尋 | 法律問答 + 法規查詢 | RAG 問答、法規搜尋、案件脈絡圖 |
| 👨⚖️ 法官專區 | 法官數位孿生 + 法官時間線 | 法官查詢、時間線、分析 |
| 🏛️ 法院數據 | 法院分析 + 判決統計 | 法院比較、趨勢統計 |
| 👔 律師服務 | 律師媒合 + 訴訟策略 + 律師數位孿生 | 律師列表、推薦、策略分析 |
| ⚙️ 系統管理 | （新） | 資料上傳、系統設定 |
